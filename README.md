# Software de Peritaje

Sistema de gestión para peritajes de la gestoría EB.

## Descripción

Herramienta para registrar, administrar y hacer seguimiento de expedientes de peritaje.

## Funcionalidades previstas

- Registro de expedientes
- Seguimiento de estado
- Generación de informes
- Gestión de clientes
